Lesson 0

What Can Claude Code Actually Do?

+
Goal: Understand what Claude Code is and what you can do with it.

The Big Picture

Claude Code is like having a brilliant, patient friend who happens to know how to build anything on a computer. You tell them what you want in plain English, and they build it — asking your permission before making any changes.

You don't need to know how to code. You don't need to understand computers deeply. You just need to be able to describe what you want.

What It Looks Like

You open the Claude Code desktop app, start a conversation, and type something like:

You: Make me a website to track my book reading list. I want to be able to add books, mark them as read, and see a nice list of everything. Use a cozy library theme with warm colors.
Claude: I'll create a book tracker website for you! I'm going to create three files: index.html, styles.css, and script.js. May I proceed?
You: Yes!
Claude: [builds the whole thing, then explains what it made]
You: Love it! Can you make the font a bit bigger and add a star rating for each book?
Claude: [updates it]

Things You Can Build

Websites & Pages

Personal sites, recipe collections, photo galleries, event pages, digital journals, portfolios

Simple Apps

Useful Tools

Fun & Creative

Lesson 1

Installing Claude Code (One-Time Setup)

+
Goal: Install Claude Code on your Windows computer. You'll open a terminal window exactly once — just to run the installer. After this, you'll use the desktop app.

This is the only time you'll need to use the "command line" (a text-based window). It sounds intimidating, but it's just one copy-and-paste step. You've got this!

Step 1: Install Git for Windows (Required First)

Claude Code needs a free tool called Git to be installed. Here's how:

1

Open your browser and go to git-scm.com/download/win

2

The download should start automatically. Run the installer when it finishes.

3

Click Next through all the screens with the default options, then click Install. Done!

Step 2: Open PowerShell (Just This Once)

PowerShell is Windows' built-in text-based window. We're using it only to install Claude Code.

4

Press the Windows key on your keyboard, type powershell, and click Windows PowerShell when it appears.

A blue or black window will open. That's PowerShell. Don't worry — you're just going to paste one thing and close it.

Step 3: Run the Installer

5

Click inside the PowerShell window, then copy and paste this command and press Enter:

irm https://claude.ai/install.ps1 | iex

You'll see text scrolling as it installs — that's normal. Wait for it to finish.

6

When it's done, close the PowerShell window. You won't need it again!

Alternative: Install with WinGet

If the above didn't work, try this instead in PowerShell:

winget install Anthropic.ClaudeCode
That's the hard part done! Claude Code installs itself and automatically stays up to date in the background. You never have to run that command again.
If you get an error: Try closing PowerShell and opening a new one, then running the command again. If you're still stuck, skip ahead to the Troubleshooting lesson.

Launching the Desktop App & Your First Project

+
Goal: Open Claude Code's desktop app, set up a project folder, and have your first real conversation with Claude.

Open the Desktop App

1

Press the Windows key and search for Claude Code. Click to open it.

You'll see a chat interface — it looks similar to other chat apps you've used.

Create a Folder for Your Project

Claude Code works inside a folder on your computer. Everything it creates goes there. Think of it as Claude's workspace.

4

Open Your Folder in Claude Code

5

6

Lesson 4

How Working with Claude Actually Goes

+
Goal: Understand the rhythm of working with Claude so you feel confident and in control.

The Basic Flow

Every session with Claude follows the same comfortable pattern:

  1. You ask for something in plain English
  2. Claude thinks about what to do and tells you its plan
  3. Claude asks permission before creating or changing any files
  4. You approve or decline each action
  5. You give feedback and refine until you're happy

You are always in control. Claude cannot do anything to your computer without your approval first.

Key Things to Know

Approving Actions

When Claude wants to create or edit a file, it will show you what it plans to do and ask. You can approve, decline, or ask Claude to change its approach.

Sending Messages

Press Enter to send. For a new line without sending, press Shift + Enter.

Stopping Claude

If Claude is doing something and you want it to stop, press Escape. To exit the app entirely, just close the window.

Starting Fresh

Type /clear to wipe the conversation and start a new task. Useful when you're done with one thing and moving on to another.

Helpful Commands You Can Type

Command What It Does
/help Shows available commands and tips
/clear Clears conversation history, starts fresh
/compact Summarizes a long conversation to save memory
/doctor Checks your installation and flags any problems

Sharing Images with Claude

One of the most powerful things you can do is show Claude a picture of what you want:

  • Paste a screenshot: Take a screenshot with Windows + Shift + S, then press Ctrl + V in the Claude chat. Say something like "make my website look like this."
  • Attach a file: Use the attach/paperclip button in the desktop app to share images, documents, or any file.
  • Drag and drop: Drag an image file directly into the chat window.
Images are super useful! Found a website design you love? Screenshot it and show Claude. Want Claude to recreate something? Show it a photo. It can see and understand images just like you can.
Lesson 5

Writing Good Prompts (Talking to Claude Effectively)

+
Goal: Learn how to describe what you want in a way that gets great results.

The better you describe what you want, the better Claude's output will be. Here are some tips:

Be Specific

Vague

"Make me a website"

Specific

"Make me a personal portfolio website with a dark theme. Include sections for About Me, My Projects, and Contact Info. Use a modern, clean design."

Describe the Look and Feel

Vague

"Make it look nice"

Specific

"Use soft pastel colors (light pink and lavender), rounded corners on buttons, and a friendly sans-serif font like the ones you see on recipe blogs"

Show Claude What You Mean

Words can only go so far. If you have a picture in mind, show it!

Just words

"I want it to look like a cozy coffee shop menu"

Words + image

Screenshot of a coffee shop menu + "Make my recipe site look like this, with this color palette and this style of typography"

Iterate and Refine

You don't have to get it perfect in one message! Build on what Claude makes:

You: Create a recipe card for chocolate chip cookies
Claude: [creates a recipe card]
You: I like it, but can you make the font bigger and add a photo placeholder at the top?
Claude: [updates the design]
You: Perfect! Now change the background to a warm cream color.

Prompt Templates to Try

For websites:

Create a [type] website about [topic]. Use [colors/style]. Include sections for [section 1], [section 2], and [section 3]. Make it mobile-friendly.

For fixing something:

The [thing] isn't working right. When I [do this], it should [expected behavior], but instead it [actual behavior]. Can you fix it?

For changes:

In the [filename] file, change [specific thing] to [what you want instead].

Lesson 8

Tips, Tricks & Troubleshooting

+
Goal: Quick reference for common issues and helpful tricks.

Common Issues & Fixes

Claude Code won't open or is not installed

Fix: Go back to Lesson 1 and run the installer command again. If it's already installed but not showing up, try restarting your computer. You can also open PowerShell and type:

claude doctor

This checks your installation and tells you exactly what's wrong.

Permission errors when installing

Fix: Open PowerShell as Administrator: press Windows, type "powershell", right-click it, and choose "Run as administrator." Then run the install command again.

Claude seems stuck or frozen

Fix: Press Escape to interrupt. If that doesn't work, close and reopen the app.

My website doesn't look right

Fix: Tell Claude what's wrong! Be specific: "The header text is too small" or "The colors don't match what I asked for." Better yet, take a screenshot of what it looks like, paste it into Claude, and say "this isn't right — can you fix it?"

Claude made a mistake and I want to undo

Fix: Just tell Claude: "Undo the last change" or "Go back to how it was before." Claude will revert its edits. You can also ask it to explain the change before making it next time.

I hit my usage limit

Fix: The Pro plan has daily usage limits. If you hit them, you'll need to wait until the next day or consider upgrading to a Max plan. This usually only happens if you're doing a lot of intensive work in a single day.

Tips That Make a Big Difference

  • One project per folder. Always create a new folder for each new thing you're building. This keeps Claude focused and your files organized.
  • Use screenshots generously. If you want something to look a certain way, showing is faster than describing. Screenshot anything — websites, apps, photos — and paste it right into the chat.
  • Paste error messages directly. If something goes wrong and you see an error, copy the whole message and paste it to Claude. It will fix it.
  • CLAUDE.md for your preferences. Create a file called CLAUDE.md in your project folder and write your preferences in it (e.g., "always use pastel colors" or "keep the code simple"). Claude reads this at the start of every session automatically.
  • Use /compact for long sessions. If you've been working with Claude for a long time in one session, type /compact to help it stay focused.
  • Be honest about what you don't like. Claude doesn't have feelings. If you don't like something, just say "I don't like that, try again with..." It won't be offended!

Frequently Asked Questions

Is Claude Code free? How much does it cost?

Claude Code is not free, but it's very affordable for casual use. Here are the current plans:

  • Pro — $20/month: The best starting plan for individuals. Great for building personal projects at a comfortable pace.
  • Max 5x — $100/month: For people who use Claude heavily every day and run into the Pro limits.
  • Max 20x — $200/month: For power users who need maximum capacity.
  • API (pay-as-you-go): Pay only for what you use, with no monthly fee. Good if you use Claude Code only occasionally.

For someone just getting started, Pro at $20/month is the right choice. Check claude.ai/pricing for the latest details.

Do I need to know how to code?

Nope! That's the beauty of Claude Code. You describe what you want in plain English, and Claude writes the code. Over time you'll naturally pick up a few concepts, but it's never required.

Is it safe? Can Claude break my computer?

Claude always asks for your permission before making changes to any file. It cannot do anything without your approval. If you're unsure about something Claude wants to do, just decline and ask it to explain first.

Do I have to use the terminal?

Only once — to install it (Lesson 1). After that, you use the Claude Code desktop app, which is a normal Windows app you open from the Start Menu. No terminal needed day-to-day.

What can I build with Claude Code?

Websites, simple apps, recipe trackers, journals, quiz games, birthday trackers, countdown pages, and more. If you can describe it, Claude can probably help build it!

What if I mess something up?

Don't worry! You can always tell Claude "undo the last change" or "go back to how it was." Working in separate project folders keeps everything isolated. And you can always delete a folder and start over — nothing is permanent.

What version of Claude does Claude Code use?

Claude Code uses Anthropic's latest and most capable models — currently Claude Sonnet 4.6 and Opus 4.6 (released early 2026). These are among the best AI models available anywhere for coding and building things.
